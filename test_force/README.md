# Test Force - Оптимизированный перебор стратегий

## Запуск

### Тестирование с известными стратегиями (быстро)
```bash
python test_force/ultra_fast.py
```

### Тестирование одного домена с полным перебором
```bash
# Отредактируйте ultra_fast.py, замените TEST_DOMAINS на ["discord.com"]
python test_force/ultra_fast.py
```

## Достигнутые оптимизации

### Тайминги (было → стало)

| Параметр | Было | Стало | Ускорение |
|----------|------|-------|-----------|
| `INIT_WAIT` (WinDivert init) | 4.0 сек | 0.1 сек | ×40 |
| `SLEEP_AFTER_KILL` | 0.3 сек | 0.02 сек | ×15 |
| `CONNECT_TIMEOUT` | 2.0 сек | 0.2 сек | ×10 |
| `TLS_TIMEOUT` | 2.0 сек | 0.2 сек | ×10 |
| `HTTP_TIMEOUT` | 1.0 сек | 0.15 сек | ×6.7 |
| `TEST_PAUSE` | 0.1-0.5 сек | 0.02 сек | ×5-25 |

### Производительность

```
Оригинал (strategy_tester.py):
  679 стратегий × 3.5 сек = 2377 сек = 39.6 минут

Ultra Fast (с известными стратегиями):
  5 известных стратегий × 0.6 сек = 3 сек + накладные = 4.7 сек

Ускорение: ×506 (для случая с известными стратегиями)
```

### Реальные результаты

| Сценарий | Домены | Время | Стратегий/сек |
|----------|--------|-------|---------------|
| Оригинал | 1 | ~40 мин | 0.3 |
| Ultra Fast (перебор) | 1 | 15.2 сек | 1.2 |
| Ultra Fast (известные) | 3 | 4.7 сек | N/A |

**Итоговое ускорение: ×506** (с 40 мин до 4.7 сек для известных стратегий)

## Как работает

1. **Monkey-patch executor** - переопределяет `start_winws_with_strategy` и `stop_winws`
2. **Агрессивные таймауты** - 0.2 сек вместо 2-5 сек
3. **Минимальные задержки** - 0.02 сек вместо 0.3-0.5 сек
4. **Socket тест** - прямой TCP + TLS без curl
5. **Early stopping** - остановка после первой рабочей стратегии
6. **Известные стратегии** - проверка рабочих конфигов перед перебором

## Режимы работы

### 1. Быстрый тест известных стратегий

Скрипт автоматически проверяет известные рабочие стратегии для каждого домена.
Если хотя бы одна работает - возвращает лучшую по времени ответа.

**Время:** ~5 сек на 3 домена

### 2. Полный перебор

Если известные стратегии не сработали, начинается полный перебор 679 стратегий.

**Время:** ~15 сек на домен (до первой рабочей)

## Добавление новых стратегий

Отредактируйте `KNOWN_STRATEGIES` в `ultra_fast.py`:

```python
KNOWN_STRATEGIES = {
    "example.com": [
        ("name1", "--dpi-desync=..."),
        ("name2", "--dpi-desync=..."),
    ],
}
```

## Ограничения

1. **WinDivert требует минимум 100ms** - при 0.1s инициализации возможны сбои
2. **Слишком агрессивные таймауты** - могут давать ложные negatives
3. **Только Windows** - использует taskkill для убийства winws
